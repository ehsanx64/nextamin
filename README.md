# laralab-front

Frontend UI for [laralab](https://github.com/ehsanx64/laralab)

## Introduction 

This frontend is a SSG web application.

## Setup

Install the packages:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
